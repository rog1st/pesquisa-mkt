## 🚀 Funcionalidades
- Coleta de Dados: Registra nome, idade e opinião (Excelente, Bom, Ruim).
- Validação de Erros: Impede que o programa trave caso o usuário digite letras no campo de idade ou opções inválidas.
- Log em Arquivo: Salva cada resposta individualmente em resultado_completo_pesquisa.txt.
- Relatório Estatístico: Calcula:
- Quantidade total por categoria.
- Porcentagem de cada resposta.
- Média de idade das pessoas que responderam "Ruim".
- Gráfico Visual: Gera uma barra de progresso em ASCII para visualizar os resultados no terminal.
